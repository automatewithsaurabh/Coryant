Brief: $ARGUMENTS

Run MISCONCEPTION_HUNT moment type. Produce:
- Error classification: CARELESS / PROCEDURAL / CONCEPTUAL / MISSING PREREQUISITE / NOTATION CONFUSION
- Root cause: state the student's actual wrong mental model precisely — not "they don't understand X" but "they believe X is [wrong belief]"
- Why this belief arises (systematic cause, not "they didn't listen")
- Diagnostic question to confirm the misconception is present
- 5-step remediation: surface → cognitive dissonance → correct model → targeted practice → residual check
- What NOT to do (the common tutoring response that doesn't fix this)

This is a diagnosis, not a re-explanation. The goal is to fix the root cause, not the surface error.
