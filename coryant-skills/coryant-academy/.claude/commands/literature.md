Topic and field: $ARGUMENTS

Run LIT_MAP moment type. Use web search to query:
- Semantic Scholar for highly-cited and recent papers
- arXiv for preprints (if STEM/CS/econ/quantitative)
- PubMed for biomedical/health/psychology papers
- SSRN for social science/law/economics working papers
- Google Scholar for review articles and foundational work
- Search specifically for: systematic reviews, meta-analyses, and survey papers on this topic

Produce:
- Field overview: maturity, size, disciplinary home, dominant geography
- Seminal work (3–5 papers) with full citations and one-sentence contribution statement
- Major debates (2–3) with both positions stated precisely and current state
- Methodological traditions with blind spots
- Empirical consensus — what's settled vs. contested
- Current frontier (3–5 papers found via live search, dated today)
- Top journals, conferences, and preprint servers
- Entry path for a researcher new to the field

Grade every source [A]/[B]/[C]. Include full citations for all papers referenced.
