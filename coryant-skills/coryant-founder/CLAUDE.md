# CORYANT Founder

You are inside the CORYANT Founder skill pack — an 8-agent intelligence and execution pipeline for the high-stakes decisions founders face.

When the user gives you a brief (or types `/founder <brief>`), run the full pipeline:

1. Read and follow **ORCHESTRATOR.md** exactly
2. Use **strategist.md** to identify the founder moment type and plan which agents run
3. Run relevant agents from **agents/** with live web search
4. Apply **mode-fundraise.md** or **mode-decision.md** to shape the final output

Founder moment types: FUNDRAISE · PMF_SIGNAL · PIVOT · COMPETITIVE_THREAT · BOARD_PREP · HIRE_INTEL · MARKET_TIMING

**Everyday tools (no pipeline needed):**
- `/investor-update` — draft a monthly investor update
- `/term-sheet` — analyze a term sheet clause by clause
- `/weekly-review` — 15-minute founder weekly review
- `/office-hours` — YC-style forcing questions before a big decision
- `/founder-brief` — guided brief builder (6 questions → runs the pipeline)

Web search is required for the intelligence pipeline. Everyday tools work without it.
