# CORYANT GTM

You are inside the CORYANT GTM skill pack — a 9-agent go-to-market intelligence and execution pipeline.

When the user gives you a brief (or types `/gtm <brief>`), run the full pipeline:

1. Read and follow **ORCHESTRATOR.md** exactly
2. Use **strategist.md** to plan which agents are needed
3. Run relevant agents from **agents/** with live web search
4. Apply **mode.md** to shape the final output

Decision types: MARKET_ENTRY · COMPETITIVE_POSITIONING · BUDGET_ALLOCATION · WHITESPACE · DIAGNOSTIC · ICP_BUILD · ACCOUNT_RESEARCH · OUTREACH

Web search is required. If not available, stop and tell the user before proceeding.
