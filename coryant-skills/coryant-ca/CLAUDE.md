# CORYANT CA

You are inside the CORYANT CA skill pack — a 7-agent intelligence pipeline for the advisory layer of CA practice: the work that determines whether you walk into a client meeting informed or improvising.

When given an engagement brief (or `/ca <brief>`), run the full pipeline:

1. Read and follow **ORCHESTRATOR.md** exactly
2. Use **strategist.md** to identify the CA work type and plan which agents run
3. Run relevant agents from **agents/** with live web search
4. Apply **mode-advisory.md** (internal report) or **mode-client.md** (client-facing document)

CA work types: CLIENT_INTEL · INDUSTRY_BENCHMARK · DEAL_SUPPORT · REGULATORY_WATCH · AUDIT_RISK · VALUATION_RESEARCH · ADVISORY_PITCH

**Everyday tools (no pipeline needed):**
- `/ca-brief` — guided brief builder (6 questions → runs the pipeline)
- `/ca-email` — draft a client email or advisory memo
- `/engagement-letter` — draft engagement letter for a new mandate
- `/meeting-prep` — pre-meeting brief: agenda, questions, watch-fors
- `/filing-deadlines` — upcoming compliance deadlines (India or UK)

Web search is required for the intelligence pipeline. Everyday tools work without it.
