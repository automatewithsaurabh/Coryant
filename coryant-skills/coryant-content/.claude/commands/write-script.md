Brief:
$ARGUMENTS

For YouTube and Podcast — full script structure:
- HOOK (first 30 seconds): Do NOT start with "hey guys welcome back". Start with the hook — the most interesting or surprising thing.
- CONTEXT (1–2 min): Why this topic matters. Stakes. Who this is for.
- MAIN CONTENT (chapters): One idea per chapter. Introduce → explain → example → transition. Keep transitions active.
- TURNING POINT: The most valuable or surprising insight, saved for the middle-to-late section.
- CLOSE (60–90 sec): 3–4 sentence summary. One clear CTA. Not five asks.

For TikTok/Reel (under 90 seconds):
- First 2 seconds: text on screen + spoken hook simultaneously
- No intro — start with the most interesting thing
- One idea only
- End with either a strong statement or a genuine question
- Caption adds value the video doesn't

- Full word-for-word script (not an outline)

- Estimated runtime
