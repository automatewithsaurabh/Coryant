Generate 10 opening hooks for the following using CORYANT Content Writer standards from `agents/content-writer.md`.

Brief:
$ARGUMENTS

Requirements:
- 10 distinct hooks using different hook types — not 10 variations of the same approach
- Cover these hook types (use all that are relevant to this niche/platform):
  1. Contrarian claim — "Everyone says X. They're wrong."
  2. Unexpected statistic — "[Number] that will change how you think about X"
  3. Vulnerable opener — "I [embarrassing failure]. Here's what I learned."
  4. Pattern interrupt — Something that breaks the reader's expected pattern
  5. Strong question — One that they genuinely don't know the answer to
  6. Bold prediction — "In 3 years, [something counterintuitive]"
  7. Insider reveal — "Here's what actually happens inside [X] that nobody talks about"
  8. Identity challenge — "If you're [identity], you've probably made this mistake"
  9. Specific story opening — One sentence that drops you into a specific moment
  10. The list promise — "The [n] things I wish I knew before [X]"

For each hook:
- Write the hook (1–2 lines maximum)
- Name the hook type
- Name the emotional trigger (curiosity / fear / identity / relief / aspiration / solidarity)
- One sentence on why it would work for this specific audience on this platform

End with: "Top 3 recommended, in order" — with a one-line reason for each.
