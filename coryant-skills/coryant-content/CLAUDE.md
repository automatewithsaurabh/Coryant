# CORYANT Content

You are inside the CORYANT Content skill pack — an 8-agent intelligence and execution pipeline covering the full content stack: from knowing exactly who your audience is and what they want, to producing content that earns attention.

When given a content brief (or `/content <brief>`), run the full pipeline:

1. Read and follow **ORCHESTRATOR.md** exactly
2. Use **strategist.md** to identify the content moment type and plan which agents run
3. Run relevant agents from **agents/** with live web search
4. Apply **mode-strategy.md** (intelligence/strategy report) and/or **mode-content.md** (ready-to-publish content)

Content moment types: STRATEGY_BUILD · COMPETITOR_CONTENT · AUDIENCE_INTEL · TREND_HUNT · CHANNEL_STRATEGY · SEO_RESEARCH · VIRAL_ANATOMY · CONTENT_AUDIT

**Everyday tools (no full pipeline needed):**
- `/content-brief` — guided brief builder (6 questions → runs the pipeline)
- `/write-post` — draft a ready-to-publish post for any platform
- `/write-newsletter` — draft a complete newsletter issue
- `/write-script` — YouTube, podcast, TikTok, or Reel script
- `/repurpose` — turn one piece of content into multiple formats
- `/hook-writer` — 10 hook variations for a topic
- `/content-calendar` — 30 or 90-day content calendar

Web search is required for the intelligence pipeline. Everyday tools work without it.
