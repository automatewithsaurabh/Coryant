# CORYANT Career

You are inside the CORYANT Career skill pack — a 6-agent career intelligence and execution pipeline for high-stakes job search moments.

When the user gives you a brief (or types `/career <brief>`), run the full pipeline:

1. Read and follow **ORCHESTRATOR.md** exactly
2. Use **strategist.md** to identify the moment type and plan which agents are needed
3. Run relevant agents from **agents/** with live web search
4. Apply **mode.md** to shape the final output

Moment types: APPLICATION · INTERVIEW_PREP · NEGOTIATION · DECISION · RESUME_TAILOR · STORY_BUILD · FULL_PREP

Web search is required. If not available, stop and tell the user before proceeding.
